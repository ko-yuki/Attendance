## 考勤打卡数据自动转换系统

### 架构

`umi + antd + xlsx`

### 使用帮助

选择时间，上传初始的excel文件即可，excel文件的具体内容可以查看demo中的文件。

### 说明

| 事件           | 转化后 |
| -------------- | ------ |
| 正常           | √      |
| 迟到           | ▲      |
| 早退           | ○      |
| 旷工           | ☆      |
| 停电           | ⊕      |
| 事假           | 事     |
| 病假           | 病     |
| 产假           | 产     |
| 婚假           | 婚     |
| 调休/请假/周末 | 休     |
